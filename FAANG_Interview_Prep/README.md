## 🔧 Setup
```bash
# Most code examples are Node.js — run them with:

# SQL examples can be run in any PostgreSQL/MySQL client
# MongoDB examples need a running MongoDB instance or Atlas
```

> **Tip:** Read the theory first, then try to write the code from memory, then check the solution.
