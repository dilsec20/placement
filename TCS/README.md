## ⚠️ Important Exam Day Instructions

1. **No eye movement downward** while giving the examination
2. **No extra rough paper** — use the on-screen rough pad
3. **Calculator available** on desktop screen
4. **All sections are mandatory** in both rounds
5. **No negative marking** — attempt every question!
6. **Non-adaptive exam** — all students get same difficulty level
