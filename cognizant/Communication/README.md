### Word Stress Rules

```
✅ Emphasize the CAPITALIZED syllable:

2-syllable NOUNS → stress on FIRST syllable
  - RE-cord, PRO-ject, PRE-sent, OB-ject, CON-tract

2-syllable VERBS → stress on SECOND syllable
  - re-CORD, pro-JECT, pre-SENT, ob-JECT, con-TRACT

Words ending in -TION/-SION → stress on syllable BEFORE
  - edu-CA-tion, com-mu-ni-CA-tion, de-CI-sion

Words ending in -IC → stress on syllable BEFORE
  - rea-LIS-tic, e-co-NOM-ic, au-to-MAT-ic

Words ending in -ITY → stress on syllable BEFORE
  - ac-TIV-i-ty, u-ni-VER-si-ty, op-por-TU-ni-ty
```

## 2. Speaking / Fluency

### 📌 What Is Tested?
- Speaking clearly and confidently
- Organizing thoughts logically
- Using appropriate vocabulary
- Avoiding fillers (um, uh, like, you know)
- Maintaining proper pace (not too fast/slow)

### Common Speaking Topics (Be Ready!)

#### Self Introduction (MOST IMPORTANT)
```
"Good [morning/afternoon], my name is [Your Name]. I'm from [City, State].
I recently [graduated from / am pursuing] [Degree] in [Branch] from [College Name].

During my academics, I developed strong skills in [Java, SQL, Web Development].
I have worked on projects like [Project 1] where I used [technologies].

My strengths include [problem-solving, teamwork, quick learning].
In my free time, I enjoy [hobbies].

I'm excited about the opportunity at Cognizant because [reason — growth, learning,
technology, company reputation]. I believe my technical skills and eagerness to
learn make me a strong fit for this role.

Thank you."
```

#### Other Common Topics
Prepare 1-2 minute responses for these:

1. **Tell me about yourself**
2. **Why Cognizant?**
3. **Describe a challenging situation and how you handled it**
4. **What are your strengths and weaknesses?**
5. **Where do you see yourself in 5 years?**
6. **Describe your favorite project**
7. **What is teamwork? Give an example**
8. **Describe a time you showed leadership**
9. **Why should we hire you?**
10. **What motivates you?**

### STAR Method (for behavioral answers)

```
S - Situation : Set the context
T - Task      : What was your responsibility?
A - Action    : What did you do?
R - Result    : What was the outcome?

Example: "Describe a challenging situation"

S: During my final year project, our team member dropped out 2 weeks before the deadline.
T: I had to take on additional backend development along with my frontend work.
A: I created a prioritized task list, spent extra hours, and learned Node.js quickly
   to complete the backend. I also coordinated with remaining teammates.
R: We submitted the project on time and received an A grade. I also gained
   full-stack development skills.
```

### Fluency Tips
| ✅ Do | ❌ Don't |
|-------|---------|
| Speak at moderate pace | Rush through answers |
| Pause briefly to think | Use "um", "uh", "like" excessively |
| Use complete sentences | Give one-word answers |
| Maintain steady tone | Speak in monotone |
| Use transition words | Jump between ideas randomly |
| Practice daily | Memorize scripts word-for-word |

### Useful Transition Words

| Purpose | Words/Phrases |
|---------|---------------|
| **Adding** | Furthermore, Moreover, Additionally, In addition |
| **Contrasting** | However, On the other hand, Nevertheless, Although |
| **Explaining** | For instance, For example, In other words, That is |
| **Concluding** | In conclusion, To summarize, Overall, Therefore |
| **Sequencing** | Firstly, Secondly, Finally, Next, Then |
| **Emphasizing** | Indeed, In fact, Particularly, Especially |

## 3. Grammar

### 📌 What Is Tested?
- Correct tense usage
- Subject-verb agreement
- Articles (a, an, the)
- Prepositions
- Active/Passive voice
- Direct/Indirect speech
- Sentence correction
- Parts of speech

### 3.1 Tenses

| Tense | Structure | Example |
|-------|-----------|---------|
| **Simple Present** | S + V1/V1s | I **write** code daily |
| **Present Continuous** | S + is/am/are + V-ing | I **am writing** code now |
| **Present Perfect** | S + has/have + V3 | I **have written** 3 programs |
| **Present Perfect Continuous** | S + has/have been + V-ing | I **have been writing** code for 2 hours |
| **Simple Past** | S + V2 | I **wrote** code yesterday |
| **Past Continuous** | S + was/were + V-ing | I **was writing** code when he called |
| **Past Perfect** | S + had + V3 | I **had written** the code before the meeting |
| **Past Perfect Continuous** | S + had been + V-ing | I **had been writing** for hours before I stopped |
| **Simple Future** | S + will + V1 | I **will write** code tomorrow |
| **Future Continuous** | S + will be + V-ing | I **will be writing** code at 5 PM |
| **Future Perfect** | S + will have + V3 | I **will have written** it by Friday |
| **Future Perfect Continuous** | S + will have been + V-ing | I **will have been writing** for 3 hours by then |

### 3.2 Subject-Verb Agreement

```
✅ Correct Rules:

1. Singular subject → singular verb
   ✅ He writes code.        ❌ He write code.
   ✅ She is a developer.    ❌ She are a developer.

2. Plural subject → plural verb
   ✅ They write code.       ❌ They writes code.
   ✅ We are developers.     ❌ We is developers.

3. "Each/Every/Either/Neither" → singular
   ✅ Each student has a laptop.
   ✅ Every developer writes tests.
   ✅ Neither option is correct.

4. "Some/Many/Few/Both/Several" → plural
   ✅ Many students have laptops.
   ✅ Few developers write documentation.

5. Collective nouns → usually singular
   ✅ The team is working hard.
   ✅ The committee has decided.

6. "Either...or / Neither...nor" → verb agrees with NEAREST subject
   ✅ Neither the teacher nor the students were present.
   ✅ Either the students or the teacher was present.

7. Subjects joined by "and" → plural
   ✅ Java and Python are popular languages.

8. Uncountable nouns → singular
   ✅ Information is important.
   ✅ Knowledge is power.
```

### 3.3 Articles (A, An, The)

```
"A" → before consonant SOUNDS
  ✅ a book, a car, a university (yoo-), a European (yoo-), a one-way street (w-)

"An" → before vowel SOUNDS
  ✅ an apple, an hour (silent h), an MBA (em-), an honest person (silent h)

"The" → specific/known thing
  ✅ The sun rises in the east. (unique things)
  ✅ The book I bought yesterday. (specific)
  ✅ The United States. (specific names)

No article:
  ✅ I love music. (general/uncountable)
  ✅ She studies at Harvard. (proper nouns, generally)
  ✅ Water is essential. (general uncountable)
```

### 3.4 Prepositions

```
TIME:
  at → specific time: at 5 PM, at noon, at night
  on → days/dates: on Monday, on 15th May, on my birthday
  in → months/years/long periods: in May, in 2026, in the morning
  for → duration: for 3 hours, for 2 years
  since → starting point: since 2020, since Monday
  by → deadline: by tomorrow, by 5 PM
  during → within a period: during the meeting

PLACE:
  at → specific point: at the bus stop, at home, at school
  in → enclosed space: in the room, in Mumbai, in India
  on → surface: on the table, on the wall, on page 5
  to → direction: go to school, travel to Mumbai
  from → origin: from India, from the office
  between → two things: between Java and Python
  among → more than two: among the students
```

### 3.5 Active and Passive Voice

```
Active:  Subject + Verb + Object
Passive: Object + be + V3 + by Subject

CONVERSIONS:

Simple Present:
  Active:  She writes code.
  Passive: Code is written by her.

Present Continuous:
  Active:  She is writing code.
  Passive: Code is being written by her.

Simple Past:
  Active:  She wrote code.
  Passive: Code was written by her.

Past Continuous:
  Active:  She was writing code.
  Passive: Code was being written by her.

Present Perfect:
  Active:  She has written code.
  Passive: Code has been written by her.

Simple Future:
  Active:  She will write code.
  Passive: Code will be written by her.

Modal:
  Active:  She can write code.
  Passive: Code can be written by her.
```

### 3.6 Direct and Indirect Speech

```
Direct → Indirect Conversion Rules:

TENSE CHANGES:
  Simple Present    → Simple Past
  Present Cont.     → Past Continuous
  Present Perfect   → Past Perfect
  Simple Past       → Past Perfect
  Will              → Would
  Can               → Could
  May               → Might

PRONOUN CHANGES:
  I → he/she
  We → they
  My → his/her
  Our → their

TIME/PLACE CHANGES:
  today → that day
  yesterday → the previous day / the day before
  tomorrow → the next day / the following day
  now → then
  here → there
  this → that
  these → those
  ago → before

EXAMPLES:

Statement:
  Direct:   He said, "I am a developer."
  Indirect: He said that he was a developer.

Question:
  Direct:   She asked, "Where do you work?"
  Indirect: She asked where I worked.

Yes/No Question:
  Direct:   He asked, "Do you know Java?"
  Indirect: He asked if/whether I knew Java.

Command:
  Direct:   She said, "Submit the report."
  Indirect: She told me to submit the report.

Request:
  Direct:   He said, "Please help me."
  Indirect: He requested me to help him.
```

### 3.7 Common Grammar Errors

| ❌ Wrong | ✅ Correct | Rule |
|---------|-----------|------|
| I didn't **went** | I didn't **go** | Don't use V2 after did/didn't |
| He **don't** like it | He **doesn't** like it | He/She/It → doesn't |
| She is **more taller** | She is **taller** | Don't combine more + -er |
| I **am agree** | I **agree** | Agree is a verb, no "am" needed |
| **Myself** Dilip speaking | **This is** Dilip speaking | Myself ≠ I/This is |
| He gave me **an** advice | He gave me **advice** | Advice is uncountable |
| I have **less** friends | I have **fewer** friends | Fewer = countable, Less = uncountable |
| **Their** going home | **They're** going home | They're = they are |
| I **could able to** do it | I **could** do it / I **was able to** do it | Don't combine both |
| One of my friend **are** | One of my friends **is** | One of → singular verb |
| He is **elder than** me | He is **older than** me | Elder = no "than", Older than |
| I **didn't knew** | I **didn't know** | Did + V1 (base form) |
| **Discuss about** | **Discuss** (no about) | Discuss is transitive |
| **Return back** | **Return** | Return already means back |
| **Revert back** | **Revert** or **Reply** | Revert = go back to original |
| **Prepone** the meeting | **Reschedule earlier** | Prepone is not standard English |

### 3.8 Sentence Correction Practice

```
❌ Each of the students have completed their assignment.
✅ Each of the students has completed his/her assignment.
   (Each → singular)

❌ The news are very shocking.
✅ The news is very shocking.
   (News → uncountable singular)

❌ Neither Ram nor his friends was present.
✅ Neither Ram nor his friends were present.
   (Nearest subject "friends" → plural)

❌ He is one of the best player in the team.
✅ He is one of the best players in the team.
   (One of the + plural noun)

❌ I have been waiting since two hours.
✅ I have been waiting for two hours.
   (Since = point in time, For = duration)

❌ He told that he was busy.
✅ He said that he was busy.
   (told + person, said + that)

❌ Despite of the rain, we went out.
✅ Despite the rain, we went out.
   (Despite = no "of", In spite of = with "of")

❌ I am working here since 2020.
✅ I have been working here since 2020.
   (Since → present perfect continuous)
```

## 4. Listening Comprehension

### 📌 What Is Tested?
- Understanding spoken English passages or conversations
- Answering questions based on what you heard
- Identifying main ideas, details, and inferences
- Understanding accents and speed variations

### Tips for Listening Section

| # | Tip |
|---|-----|
| 1 | **Read questions first** before the audio plays (if visible) |
| 2 | **Focus on keywords** — who, what, where, when, why, how |
| 3 | **Don't panic** if you miss a word — focus on overall meaning |
| 4 | **Take quick notes** — jot down numbers, names, key facts |
| 5 | **Listen for signal words** — "however", "therefore", "in conclusion" |
| 6 | **Pay attention to tone** — it reveals attitude (positive, negative, neutral) |
| 7 | **Eliminate wrong options** — use logic if unsure |
| 8 | **Practice daily** — listen to English podcasts, news, TED talks |

### Types of Listening Questions

| Type | What to Focus On |
|------|-----------------|
| **Main Idea** | "What is the passage about?" → Listen for the opening/closing |
| **Detail** | "What time...?" "How many...?" → Listen for specific facts |
| **Inference** | "What can we conclude?" → Read between the lines |
| **Vocabulary** | "What does X mean?" → Understand from context |
| **Tone/Attitude** | "How does the speaker feel?" → Happy, concerned, neutral |
| **True/False** | Listen carefully for exact wording |

### Sample Listening Practice

**Instructions**: Read this passage aloud or have someone read it, then answer:

```
"The IT industry in India has grown significantly over the past two decades.
Companies like Cognizant, Infosys, and TCS have become global leaders in
providing technology services. The industry employs over 5 million people
and contributes approximately 8% to India's GDP. However, with the rise of
automation and artificial intelligence, the industry is rapidly transforming.
Employees now need to continuously upskill to remain relevant. Cloud computing,
data science, and cybersecurity are among the most in-demand skills today."

Questions:
1. How many people does the IT industry in India employ? → Over 5 million
2. What percentage does IT contribute to India's GDP? → Approximately 8%
3. What is causing the industry to transform? → Automation and AI
4. Name three in-demand skills mentioned. → Cloud computing, data science, cybersecurity
5. What do employees need to do to stay relevant? → Continuously upskill
```
