## 💡 Tips for the Assessment

1. **Read the question carefully** before writing code
2. **Start with easy questions** first to build confidence
3. **For Java**: Focus on correct syntax, edge cases, and proper exception handling
4. **For SQL**: Practice writing queries by hand; understand JOIN types thoroughly
5. **For Web UI**: Know CSS selectors, JS DOM manipulation, and event handling
6. **Don't spend too much time** on one question — move on and come back later
7. **Test edge cases**: empty arrays, null strings, boundary values
8. **Use proper naming conventions** in your code

## 📚 Quick Reference

### Java Key Topics
- OOP (classes, inheritance, polymorphism, abstraction, encapsulation)
- Collections (ArrayList, HashMap, HashSet, LinkedList)
- String methods (charAt, substring, indexOf, split, replace)
- Exception handling (try-catch-finally, throws, custom exceptions)
- Date & Time API (LocalDate, LocalTime, LocalDateTime)
- Sorting & Searching algorithms

### SQL Key Topics
- DDL: CREATE, ALTER, DROP, TRUNCATE
- DML: INSERT, UPDATE, DELETE, SELECT
- DCL: GRANT, REVOKE
- TCL: COMMIT, ROLLBACK, SAVEPOINT
- JOINs: INNER, LEFT, RIGHT, FULL OUTER, CROSS, SELF
- Subqueries, Views, Indexes, Functions

### Web UI Key Topics
- HTML5 semantic elements, forms, tables
- CSS selectors, box model, flexbox, grid, animations
- JavaScript DOM, events, ES6+, array methods
